NWERC15 B. Better Productivity

Posted on 2019-04-11

给定 $n\le 200$ 个区间 $[l_i,r_i] $，将它们分为恰好 $p\le n$ 组，使得每组组内区间的交非空，且每组的交的长度之和最大。

Observation: 如果一个区间包含另一个区间，那么它要么单独成组，要么对答案不产生影响。这是因为，假如它不单独成组的话，把它和一个并不被它包含的区间放在一组，有可能会使得那一组答案变劣，而如果把它和一个被它包含的区间放在一组，则答案一定不会变劣，且不考虑它不会对答案产生影响。

因此，我们先把所有包含其他区间的区间去掉。把剩下的区间按左端点升序排序之后右端点也是升序的，这样的话，每一组一定是连续的一段区间（证明略）。这样就可以 dp 了。

$f_{i,j}$ 表示把前 $j$ 个区间分成 $i$ 组的最大长度和，$f_{i,j} = \max { f_{i-1,k}+len(k+1,j) }$，其中 $len(l,r)$ 表示第 $l$ 个到第 $r$ 个区间的交的长度。

求出 dp 数组之后再考虑 $p$ 个分组中有几个是之前去掉的区间就行了。

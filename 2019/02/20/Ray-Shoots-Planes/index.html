Ray Shoots Planes

Posted on 2019-02-20

Given $n \le 1111$ planes, find the maximum number of planes that can be hit by a ray starting from origin.

Any three planes have exactly one common point.

It can be observed that where a plane is does not matter as long as it’s on the same side of the origin.

So we can represent a plane $P$ by its normal vector $v$ ($v$ should be starting from origin and directing at $P$). A ray $r$ can hit $P$ means that the angle between $v$ and $r$ is less than $\dfrac{\pi}{2}$.

Similarly, we can represent the shooting ray $r$ by the plane whose normal vector is $r$.

In that way, the problem is transformed into finding a plane that has the most number of points (endpoints of the normal vectors) above.

It’s easy to notice that instead of finding the plane $P​$ with most number of points above it, we can try to find the plane with most number of points above or on it. Because we can rotate the plane slightly to make the points on it move to above.

We already know that the origin $O$ is on the plane $P$. Choose another point $T$ from the given points ($n$ possiblities). Now the possible plane is rotating around the axis $OT$, if we look along $OT$ (projecting the points on a plane that’s perpendicular to $OT$), the plane $P$ looks like a line, and points that are above $P$ are above the line.

So now we need to solve the 2-dimensional problem of finding a line with the most number of points above it, which can be done with clockwise sort and two-pointers technique in $O(n\log n)$ time.

The total complexity is $O(n^2\log n)$.
